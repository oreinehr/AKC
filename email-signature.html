How to install

  1. Click Copy for email above.
  2. Gmail: Settings → See all settings → General → Signature → Create new → paste with ⌘ Shift V (Mac) or Ctrl Shift V (Win) to paste formatted.
  3. Apple Mail: Paste into a new compose window, select all, drag into Mail → Settings → Signatures.
  4. Outlook: File → Options → Mail → Signatures → New → paste into the rich-text editor.
  1. Font note: email clients don’t load web fonts. This version renders in Georgia (serif) and Arial — fallbacks chosen to preserve the brand character.
  2. Image note: no logo in this version — avoids image-blocking issues in enterprise clients. A logo variant can be added once a hosted URL for the SVG is confirmed.
